# Mend — Self-Reflection & Emotional Healing Platform
Mend is a reflective storytelling and emotional healing platform built with Next.js + Supabase + OpenAI, connected to Bayview Arts & Framing Studio (Mornington Peninsula, Victoria).

## Purpose
- Help people **frame their stories** through guided prompts and creative artefacts.
- Integrate with Bayview Arts (gallery, framing) and the Market Garden Project (community wellbeing).

## Planned structure
- mend-platform — docs + roadmap (this repo)
- mend-web — front-end (Next.js UI)
- mend-api — Supabase + OpenAI services
- mend-stories — templates & schemas
- mend-assets — brand / images
- mend-ops — CI / deployment
